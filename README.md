# ts-practice

Some ts code, **use at your own risk**.

## Requirements

Please make sure you have the latest version of node installed. [nvm](https://github.com/nvm-sh/nvm) is a solution for you.

## Scripts

```bash
# use latest version of node
nvm install

# install dependencies
yarn install

# run
yarn start

# test
yarn test

# lint
yarn lint

# lint fix
yarn lint:fix
```
